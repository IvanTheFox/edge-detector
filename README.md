# Информация
В современном мире компьютерное зрение играет значительную роль. Оно используется в широком спектре программного обеспечения: беспилотный транспорт, медицина (диагностика), промышленность (контроль качества), системы безопасности и многое другое. В связи с этим, я решил исследовать обнаружение контуров изображения с помощью компьютерного зрения, что позволяет сокращать количество информации, которое несёт изображение

В данной работе рассмотрены градиентный метод и метод Лапласа. Работа из себя представляет приложение, позволяющее загрузить изображение с компьютера пользователя и сгенерировать контурную карту этими двумя методами. Также программа позволяет менять некоторые параметры конвертации.
